# Larpercrate
Larpercrate is a fan-made demon list project inspired by Pointercrate, built entirely with HTML.

## Features
- Custom demon list pages (featuring pointercrate demons)
- Ranking pages (planned)
- Individual demon pages
- Browser-based website

## Built With
- HTML
- Base64 encoded assets/data

## Status
Active development

## Planned
- Backend/database integration
- API improvements
- More ranking features
